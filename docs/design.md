# 阿姐终端

## 项目描述

基于`NestJS(monorepo)`微服务架构搭建基础设施平台。项目采用一服一库部署和租户逻辑隔离相结合的设计方式，旨在打造一个高内聚、低耦合的应用生态圈。

后续所开发的所有应用，都可接入到平台；最终实现全平台统一的身份认证、集约化的`API`服务调用以及跨应用的数据互通。

## 架构特性

- 微服务架构: 一服一库部署，租户逻辑隔离，全平台统一身份认证。

- 全方位安全: 基于`JWT`、`OAuth2.0`认证协议以及`RBAC`、`ABAC`访问控制，精细化控制管理。

- 现代化技术: `NestJS(monorepo)`、`PostgreSQL(Prisma)`、`Redis`、`MQ`、`Kubernetes`、`Docker`+`Git(CI/CD)`等。
